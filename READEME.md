# 📘 Internship Summary – Insilicogen (Bioinformatics Training)

## 🧾 Overview
This repository contains a summary of my internship training at **Insilicogen**, focusing on bioinformatics workflows, genomic data analysis, and computational research environments.

The internship was conducted as part of a joint program:
> Post-Genome Multi-Ministry Joint Internship Program

---

## 📅 Duration
- July 1, 2019 – August 9, 2019 (6 weeks)

---

## 🧪 Training Contents

### 🔹 Week 1 – Introduction & RNA Analysis
- Orientation and internal workflow overview  
- Genomic analysis using CLC Main Workbench  
- RNA Structure & Workflow analysis  
- Presentation and feedback  

---

### 🔹 Week 2 – Metagenomics
- Genomic analysis using CLC Genomic Workbench  
- Metagenomics data analysis  
- Presentation and feedback  

---

### 🔹 Week 3 – Computational Environment
- Linux basics and command-line usage  
- Python basics  
- Virtual environment setup using Bioconda  
- Jupyter Notebook introduction  

---

### 🔹 Week 4 – Statistical Analysis (R)
- NGS data analysis concepts  
- RNA-seq & Epigenomics overview  
- R programming fundamentals  
- Data analysis and visualization using R  

---

### 🔹 Week 5 – Database & System Design
- Database system concepts  
- Homepage improvement idea proposal  
- User story, prototype, and flowchart design  
- SQL-based database implementation  

---

### 🔹 Week 6 – GUI-Based DB System
- GUI-based database system development using Neo4j  
- Business Canvas Model (BCM) design  
- Final presentation and feedback  

---

## 🧠 Key Skills Acquired
- Bioinformatics workflow understanding  
- NGS data analysis (RNA-seq, Metagenomics)  
- Linux & Python basic usage  
- R-based statistical analysis and visualization  
- Database design and implementation (SQL, Neo4j)  
- Data-driven presentation and communication

## 💬 Reflection
Although I had prior experience in biological data analysis, this internship strengthened my understanding of **bioinformatics workflows and computational tools**.

It also highlighted the importance of integrating **data analysis, programming, and system design** in real-world research environments.
